This project predicts the selling price of a car using Machine Learning.

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Algorithm
- Linear Regression

## Features
- Data Preprocessing
- Train-Test Split
- Model Training
- Price Prediction
- Performance Evaluation

## Results
The model predicts car prices based on vehicle features and historical data.
